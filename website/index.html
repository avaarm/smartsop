Alternative Installation Methods

# Run the full stack with Docker Compose
git clone https://github.com/smartsop/smartsop.git
cd smartsop
docker compose up -d

# Open http://localhost:4000
# 1. Clone and install
git clone https://github.com/smartsop/smartsop.git
cd smartsop
npm install
pip install -r requirements-gmp.txt

# 2. Install Ollama and pull a model
brew install ollama          # macOS
ollama pull llama3

# 3. Start backend + frontend
python3 gmp_server.py &     # port 5001
npm start                    # port 4200

Get Running in 3 Minutes

1

Install Ollama

Download Ollama and install it. Then pull the Llama 3 model:

ollama pull llama3
2

Download SmartSOP

Download the installer for your platform above and run it. On macOS, drag to Applications. On Windows, run the installer. On Linux, make the AppImage executable.

3

Start Building

Launch SmartSOP. Pick a template, fill in your product details, and let the AI generate each section. Export to DOCX when ready.
